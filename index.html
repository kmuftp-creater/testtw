系統說明與教學手冊

🚀 快速上手三步驟

  1. 單股快速健檢： 如果您手上有特定股票想診斷，直接在最上方的「🔍單股快速健檢」輸入代號即可，系統會瞬間給出圖表與評分！
  2. 條件篩選： 切換台股或美股，並勾選您想要的條件（所有條件為 AND 嚴格比對），點擊「開始篩選」。
  3. 查看 AI 診斷： 掃描完成後，點擊列表中的「AI 診斷」按鈕，即可觀看視覺化走勢圖表與白話文的進出場建議。

💡 系統特色與核心功能

  1. 健康度評分 (0~100)： AI 會根據站上均線、RSI、量能等因子計算總分，讓您一眼看穿股票體質。
    🟢 80分以上 (體質優良)：多頭強勢，各項指標均表現優異。
    🟡 50~79分 (震盪盤整)：多空交戰，方向尚未完全明朗，需等待突破訊號。
    🔴 49分以下 (弱勢危險)：空頭弱勢，跌破重要均線或動能極弱，接刀風險極高。
  2. 全市場真實籌碼 (TWSE OpenAPI)： 系統在您點擊篩選時，會極速向臺灣證交所官方 API 抓取全市場「三大法人買賣超」數據，提供您最真實的外資與投信動向（註：目前官方 API 主要支援上市股票）。
  3. 支撐壓力線自動繪製： AI 診斷圖表會自動畫出「近20日最高價(壓力)」與「近20日最低價(支撐)」，並計算空間比例輔助您的盈虧比判斷。
  4. 多因子綜合評分 (防洗盤)： 勾選「適合買進」時，系統強制要求股價必須站上 MA20 月線，且多項指標加總需達標，幫您過濾假突破。
  5. 一鍵清除功能： 點擊「🗑️ 一鍵清除」按鈕，可快速重置所有篩選條件、清除自選股名單與檢測結果，方便您重新開始分析。

📖 關鍵術語與參數說明

  • MA5 (周線) / MA20 (月線)： 過去 5 天與 20 天的平均收盤價。站上月線代表中期趨勢轉強，跌破代表轉弱。
  • RSI (相對強弱指標)： 衡量股價漲跌力度。低於 30 為超賣 (容易反彈)，高於 70 為超買 (容易回檔)。
  • 量比： 今日成交量與過去 5 日平均成交量的比值。大於 1.5 倍代表有異常資金動能或主力介入。
  • IBS (內部強弱指標)： 衡量當日收盤價在當日震盪區間的相對位置。數值越低 (≤0.2) 代表收在最低點附近，隔日容易有技術性反彈；數值越高 (≥0.8) 代表收在最高點附近，容易遇到獲利賣壓。
  • KD (隨機指標)： 衡量近期價格強弱。K線上穿D線稱為「黃金交叉」(短線動能轉強)；K線下穿D線稱為「死亡交叉」(短線動能轉弱)。
  • K線 (Candlestick)： 紀錄一天股價走勢的圖形。實體很長的紅K(大陽線)代表買盤強勢；實體很長的黑K(大陰線)代表賣壓沉重。帶有長上影線代表遭遇反壓，長下影線代表下方有支撐。
  • 停利與停損是如何計算的？
    系統採用固定比例法（適合短波段交易）：
    • 停利目標1： 收盤價 + 5% (適合短線先入袋為安)
    • 停利目標2： 收盤價 + 10% (波段目標)
    • 建議停損： 收盤價 - 5% (跌破此價格強烈建議認賠出場)

⚔️ 實戰範例教學

【範例一】跟著主力買進： 勾選「適合買進」+「投信買超」+「外資買超」。
👉 適用時機：尋找技術面強勢且具有真實籌碼護航的安心標的。

【範例二】搶反彈策略 (左側交易)： 勾選「超跌反彈」+「IBS ≤ 0.2」+「跌破 MA20」。
👉 適用時機：個股短線乖離過大，尋找收在當日最低點且 RSI 極低的標的，博取隔日反彈。
